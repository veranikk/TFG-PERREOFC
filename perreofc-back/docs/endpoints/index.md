# API Endpoints Documentation

Welcome to the PerreoFC API documentation. Below is the list of available endpoint categories:

- [Authentication](auth.md) - Login, register, and password management.
- [User Profiles & Admin](users.md) - Profile management, settings, and administration.
- [Teams](teams.md) - Team details, squads, and statistics.
- [Players](players.md) - Player profiles, performance, and match history.
- [Matches](matches.md) - Match details, lineups, events, and technical stats.
- [Bets](bets.md) - Prediction system and betting management.
- [Classification](classification.md) - League standings.
- [Top Scorers](top_scorers.md) - Goal rankings.
- [Rankings](rankings.md) - User leaderboards.
- [Points](points.md) - Points history and daily rewards.
- [News](news.md) - Articles and updates.
- [Events](events.md) - Social and league events.
- [Notifications](notifications.md) - User notification system.
- [MVP Votes](mvp_votes.md) - Match MVP voting system.
- [Seasons & Competitions](seasons_competitions.md) - League structure.
- [Groups](groups.md) - Competition groups.
- [Home](home.md) - Dashboard data summary.
- [Health](health.md) - API status check.
- [Images](images.md) - Player, staff, and news image management.
- [Upload](upload.md) - File upload to Supabase Storage (multipart and from URL).
- [Chat](chat.md) - AI agent chat (Perreito) with session management.
