# Group Endpoints

Endpoints related to competition groups.

## List Groups
Returns a list of groups.

- **URL:** `/groups`
- **Method:** `GET`
- **Auth required:** Yes

---

## Get Group Detail
Returns a specific group by ID.

- **URL:** `/groups/:id`
- **Method:** `GET`
- **Auth required:** Yes
